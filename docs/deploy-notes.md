# Deploy Notes

## Option 1: GitHub Pages
1. Push your project files to a **public GitHub repo**, e.g. `us-metro-zillow-widget`.
2. Navigate to:
   **Settings → Pages → Source:** `Deploy from a branch`
   - **Branch:** `main`
   - **Folder:** `/ (root)`
3. Wait 1–2 minutes.
